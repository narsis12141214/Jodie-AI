# Blog Pre-Publish Checklist
**Jodie AI — SEO Agent**
*This checklist must be completed and confirmed before any blog post goes live on hadiphotographylondon.com. No post is approved without all items checked.*

---

## 1. CATEGORY
- [ ] Post is assigned to a relevant category (not Uncategorized)
- [ ] Category is consistent with existing taxonomy on the site

## 2. INTERNAL LINKS
- [ ] Minimum 3 internal links present in the body copy
- [ ] All links are contextually natural (not forced)
- [ ] At least one link points to a core service page (elopement, engagement, proposal, pricing)
- [ ] At least one link points to a related blog post (if one exists)
- [ ] Contact page is linked in or near the CTA

## 3. SEO BASICS
- [ ] Meta title written — includes target keyword — under 60 characters
- [ ] Meta description written — includes target keyword — under 160 characters
- [ ] H1 matches or closely reflects the target keyword
- [ ] URL slug is clean, lowercase, hyphenated, and keyword-rich (no auto-generated gibberish)
- [ ] Target keyword appears in the first 100 words of the post

## 4. IMAGES
- [ ] Every image has descriptive alt text (not filename, not blank)
- [ ] Alt text includes location or keyword where natural
- [ ] No image is missing or broken

## 5. BRAND POSITIONING
- [ ] "Cinematic" language used only when it genuinely earns its place — when a client has used it, or the work being described truly carries that quality. Not as a default descriptor or filler.
- [ ] Copy centres the couple, not the photographer
- [ ] Nervousness-to-ease transformation is present where relevant
- [ ] No generic photographer-led language (e.g. "passion for storytelling", "capturing memories")

## 6. CTA
- [ ] A clear call to action is present at or near the end of the post
- [ ] CTA links to the contact page or a relevant service page
- [ ] CTA language is direct and benefit-led (not "feel free to get in touch")

## 7. STRUCTURE
- [ ] Table of contents present (for posts over 1,000 words)
- [ ] Headings follow logical H2 — H3 hierarchy
- [ ] No walls of text — paragraphs broken into readable chunks
- [ ] Post has a clear intro that challenges or reframes the conventional assumption

## 8. FINAL CHECKS
- [ ] Post previewed on mobile before publishing
- [ ] No placeholder text or draft notes left in the copy
- [ ] Author set to Hadi Yazdani
- [ ] Publish date is correct
- [ ] Post added to internal linking plan — update any existing posts that should now link to this one

## ENFORCEMENT

Effective 13 April 2026. This checklist is a mandatory gate for every blog post.

**Process:**
1. SEO agent completes the draft
2. SEO agent runs through every item on this checklist
3. SEO agent completes the sign-off section (post title, target keyword, internal links, category, name, date)
4. Completed checklist is submitted alongside the draft, not separately, not after
5. Operator reviews and approves. No post goes live without Operator sign-off on a completed checklist.

**Why this exists:**
Blog #6 (pre-wedding photoshoot London) went live with insufficient internal links and an underplayed core insight. Both were caught and fixed post-publish. This checklist prevents that from happening again.

**Most likely failure point:** Internal linking. Minimum 3 links, at least one to a core service page, at least one to a related blog post. Check these are contextually natural, not forced.

**Brand positioning gate:** No "cinematic" as filler, no photographer-led copy, no generic phrases. Nervousness-to-ease transformation must be present where relevant. This applies to every post regardless of topic.

**Routing:** Any questions route through Operator first.
