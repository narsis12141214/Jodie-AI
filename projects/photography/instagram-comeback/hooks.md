# @hadyyazdani — Content Hooks

These 10 hooks are to be used across Instagram content wherever they fit naturally.
Do not force them. Use when the hook genuinely serves the post topic.

---

## The 10 Hooks

1. No one talks about this.
2. You're not going to like this, but —
3. Here's the truth about [topic]:
4. This is why you will never, ever [outcome].
5. If I had to start over —
6. This will change how you think about [topic].
7. I learned this way too late —
8. It cost me [thing/price/time] —
9. Nobody is going to give you permission, so I will.
10. There's one thing about [topic] that changes everything.

---

## Usage Rules

- Use in Reels as the opening line (text overlay or spoken)
- Use in single image captions as the first line — must appear before the fold (before "more")
- Use in carousel Slide 1 as the hook text
- Fill in [topic], [outcome], [thing] with something specific to Hadi's world — elopements, proposals, couple shoots, the experience of being photographed
- Never use more than one hook per post
- Alternate with non-hook openings so the account doesn't feel formulaic
