## What we're rebuilding

A complete brand-to-booking system for Hadi Photography London, using the Maddie Mae methodology as canonical source. This covers:

- Brand (vision, mission, values, specialty, ICA, UVP, voice, story)
- Website (homepage, about, contact, pricing — all pages)
- Email workflow (inquiry-to-booking, automated and manual)
- Pricing (guide PDF for couples + pricing page)
- Content strategy (blog roadmap mapped to client journey)
- Post-booking workflow (planning + timeline questionnaires)
- Site audit + technical SEO

This is not a weekend project. It's a 10-12 week rebuild minimum, sequenced so that the work each week depends only on what was locked in prior weeks.

## Phased plan

### PHASE 1 — Brand Foundation (Weeks 1-3, gated by Hadi's time)

Goal: Lock Hadi's vision, mission, values, specialty, primary ICA, UVP, brand voice (1 of 4), and brand story.

**Week 1**
- Read `Brand - Creating Your Irresistible Brand.md` (Hadi, ~2-3 hours)
- Complete Vision Statement exercise (~30 min)
- Complete Mission Statement exercise (~30 min)
- Identify Core Values exercise (~30 min)
- Start 4 Lists exercise (most/least fulfilling sessions; most/least valued clients) (~1 hour)

**Week 2**
- Finish 4 Lists exercise + identify themes (~1 hour)
- Brainstorm Specialty (~1 hour)
- Draft Specialty Statement (~30 min)
- Define Target Market (~30 min)
- Profile Primary Ideal Client Avatar (~1 hour)
- Send past client interview requests to 3 best past clients (~30 min)
  - Use template: `Brand - Interviewing Past Clients Email & Interview Template.md`
  - Target: Mozhgan, plus 2 others Hadi picks based on session feel + review quality

**Week 3**
- Conduct 3 past client interviews (record, transcribe) (~3 hours total)
- Synthesise interview findings → confirm or refine ICA + Specialty + UVP draft (~2 hours)
- Draft Unique Value Proposition (~1 hour)
- Choose Brand Voice (Straight No Chaser / Refined Dreamer / Client BFF / Hype Human) (~30 min)
- Draft Brand Story using StoryBrand framework (~1.5 hours)
- **PHASE 1 DELIVERABLE:** All brand foundation elements written and locked in a single brand brief document. Jodie produces this document from Hadi's exercise outputs.

Once Phase 1 is locked, everything downstream accelerates. **Until Phase 1 is locked, do not touch the website or email rewrites.** Doing so would lock in copy against an unlocked brand and require redoing the work.

### PHASE 2 — Website Rebuild (Weeks 4-6)

Goal: Rebuild homepage, about, contact, pricing pages in locked voice against locked UVP.

**Week 4 — Homepage**
- Strategist produces brief using `Market - Home Page Copy _ Flow Template.md` + locked brand foundation
- Copywriter drafts homepage in locked voice
- Run banned-word audit (Maddie Mae list + Hadi's existing brand rules)
- Hadi reviews, revises, signs off
- SEO agent embeds meta title + meta description aligned with new copy
- Showit/WordPress update

**Week 5 — About page + Contact page**
- Strategist briefs from About + Contact templates
- Copywriter drafts both
- Hadi reviews
- Rebuild contact form per audit checklist (form structure, value-creation question, personal question, redirect to thank-you page)
- Build thank-you page

**Week 6 — Pricing page + pricing guide PDF**
- Strategist briefs using `Market - Pricing Page Copy _ Flow Template.md` and `Price - Pricing Guide Template.md`
- Reconcile new pricing-page structure with The Presence Method commissions (The Vow, The Bloom, The Glow, The Odyssey)
- Generate pricing guide PDF for couples (Python script following the heny-aesthetic-clinic-agreement.py pattern)
- Hadi reviews, signs off, publishes

### PHASE 3 — Email Inquiry-to-Booking Workflow (Weeks 7-8)

Goal: Build the complete email workflow from contact form to booking, in locked voice.

**Week 7**
- Strategist briefs full workflow using `Book - Elopement Inquiry-to-Booking Email Templates - All Brand Voices.md` + `Wedding Inquiry-to-Booking Email Templates - All Brand Voices.md`
- Reference real inquiries from `Initial Inquiry Examples.pdf` for personalisation examples
- Copywriter drafts (in locked voice):
  - Automated response to inquiry submission
  - Initial inquiry email response (Version 1 longer, Version 2 shorter)
  - Initial inquiry text message
- Hadi reviews, signs off

**Week 8**
- Copywriter drafts (in locked voice):
  - 1st inquiry follow-up
  - 2nd inquiry follow-up
  - 3rd & final inquiry follow-up
- Operationalise: load into Dubsado or wherever the workflow lives
- Test end-to-end with a self-submitted inquiry

### PHASE 4 — Content Strategy + Blog Roadmap (Weeks 9-10)

Goal: Restart blog cadence with a strategic content roadmap mapped to client journey stages.

**Week 9**
- Strategist produces content brief using `Market - Content Idea Bank.md` (5-stage client journey: Pre-engagement → Ideas & Inspiration → Highest Priority Decisions → Executing the Vision → Filling in the Details)
- Use `Market - Content Marketing Traffic Estimator` to model traffic-to-revenue requirements
- Lock content roadmap: 12-15 prioritised topics, mapped to client journey stages and SEO keyword opportunities
- Use existing SEO data (true site totals now that pipeline v2 is running) + the 5 May audit's scored ideas (micro wedding 14/15 done; Marylebone Town Hall 12/15; wedding photographer cost 12/15; what to wear 12/15; intimate wedding 11/15; London elopement ideas 11/15; Indian wedding photographer 11/15)

**Week 10**
- Hadi commits to a blog cadence (target: 1 post per week minimum, matching the 60-day goal)
- SEO agent + blog-copywriter produce blog #8 using `Market - High-Ranking Content Creation Template + Outline.md`
- Publish blog #8

### PHASE 5 — Post-Booking Workflow (Weeks 11-12)

Goal: Build the booked-client experience workflow that turns sessions into great photographs and great reviews.

**Week 11**
- Strategist briefs questionnaire workflow using:
  - `Create - Elopement Initial Planning Questionnaire Template.md`
  - `Create - Elopement Timeline Questionnaire Template.md`
  - `Create - Wedding Timeline Questionnaire Template.md`
- Operator implements questionnaires in Dubsado
- Test with one booked client

**Week 12**
- Full site audit using `Market - Website Audit Checklist.md`
- Fix any remaining audit failures
- Lock the rebuilt photography system end-to-end
- Document what's working / what needs another round

```
PHASE 1 (Brand Foundation)
    ↓ (Vision/Mission/Values/Specialty/ICA/UVP/Voice/Story all locked)
PHASE 2 (Website Rebuild)
    ↓ (Pages live with locked voice + UVP)
PHASE 3 (Email Workflow)
    ↓ (Email scripts in locked voice)
PHASE 4 (Content Strategy)
    ↓ (Blog roadmap aligned to ICA + voice)
PHASE 5 (Post-Booking Workflow)
```

You cannot skip Phase 1. Every downstream phase fails if the brand foundation isn't locked.

You CAN run some phases in parallel once Phase 1 is locked — for example, Phase 2 (website) and Phase 3 (email scripts) can overlap. But Phase 1 is single-threaded and gated by Hadi's time on the exercises.
